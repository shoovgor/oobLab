## Төслийн тухай

Энэхүү төсөл нь C++ хэл дээр объект хандлагат програмчлалын дараах ойлголтуудыг хэрэгжүүлсэн лабораторийн ажил юм.

- `this` хувьсагч
- Static гишүүн өгөгдөл
- Static гишүүн функц
- Удамшил
- Полиморфизм
- Хийсвэр класс
- Жинхэнэ хийсвэр функц
- Дүрсүүдийг талбай болон периметрээр эрэмбэлэх

Өмнөх лабораторид хийсэн `Shape`, `Circle`, `Square`, `Triangle` классуудыг сайжруулж, static object counter болон abstract class ашигласан.
