
## Төслийн товч танилцуулга

Энэхүү лабораторийн ажил нь C++ хэл дээр объект хандлагат програмчлалын зарим чухал ойлголтуудыг хэрэгжүүлсэн төсөл юм. Өмнөх лабораторид хийсэн `Shape`, `Circle`, `Square`, `Triangle` классуудыг сайжруулж, `this` хувьсагч, static гишүүн өгөгдөл, static гишүүн функц, хийсвэр класс, жинхэнэ хийсвэр функц болон полиморфизм ашигласан.

## Гол ойлголтууд

- `this` хувьсагч
- Static гишүүн өгөгдөл
- Static гишүүн функц
- Удамшил
- Полиморфизм
- Хийсвэр класс
- Жинхэнэ хийсвэр функц
- Дүрсүүдийг талбай болон периметрээр эрэмбэлэх
